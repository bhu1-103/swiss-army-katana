# Karaoke Engine Controls

## Navigation

| Key | Action |
| :--- | :--- |
| **`Space`** | Go to the next lyric line |
| **`Down Arrow`** | Go to the next lyric line |
| **`Backspace`** | Go to the previous lyric line |
| **`Q`** | Toggle between Hiragana and Kanji lyrics |

## Music & Playback

| Key | Action |
| :--- | :--- |
| **`Enter`** | Toggle play/pause|
| **`R`** | Restart music from the beginning |
| **`Left Arrow`** | Seek music backward 5 seconds |
| **`Right Arrow`** | Seek music forward 5 seconds |
| **`Mouse Click/Drag`** | Click or drag on the bottom seek bar to pan |

## General

| Key | Action |
| :--- | :--- |
| **`Esc`** | Quit the application |
